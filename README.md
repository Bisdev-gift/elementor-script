# Elementor Snippets (Bundle)

Panggil di Elementor (Custom Code / HTML widget):

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/USERNAME/REPO@main/dist/elementor-snippets.css?v=1">
<script defer src="https://cdn.jsdelivr.net/gh/USERNAME/REPO@main/dist/elementor-snippets.js?v=1"></script>

Catatan:
- Kalau update file tapi belum berubah, naikkan angka ?v= (misal v=2).
- File dist sudah gabungan dari semua snippet agar request lebih sedikit.
